# Hub de Packs Grátis

Mini landing page hub para distribuição de packs de conteúdo (Canva, CapCut) com painel administrativo integrado ao Supabase.

## Tecnologias
- HTML5 / CSS3 (Design Moderno e Responsivo)
- JavaScript Vanilla
- Supabase (Database & Auth)

## Como rodar localmente
1. Clone o repositório
2. Abra o `index.html` ou use um servidor local (ex: `npx serve .`)
3. Configure suas chaves do Supabase em `js/config.js`

## Deploy
Desenvolvido para ser hospedado na Vercel ou Netlify.
